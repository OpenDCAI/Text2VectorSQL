## 关键问题与解决方案

### 问题1: virtiofs 文件系统导致 SQLite I/O 错误

**现象**: 所有数据库访问报 "disk I/O error"

**原因**: virtiofs 文件系统不支持 SQLite 的文件锁定机制

**解决方案**: 
```bash
# 将数据库复制到本地文件系统
bash fix_virtiofs_issue.sh
```

### 问题2: UniVectorSQL 模型需要访问权限

**现象**: 下载模型时报 401 错误

**原因**: 模型仓库是 gated repository

**解决方案**: 
1. 访问 https://huggingface.co/dongwenyao/Text2Vecsql
2. 申请访问权限（现已改为 public）
3. 修改 `download_model.py` 移除 `token=True` 参数
